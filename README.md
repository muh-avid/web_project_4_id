## Deskripsi
* Bisa menambahkan menghapus card images

* Memiliki tombol yang berfugsi :
+ Mengedit profile yaitu mengubah nama dan tentang diri
+ Menambah Foto cards
+ Hapus Cards
+ Like Cards
